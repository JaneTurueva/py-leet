Command-line tool and library for leeting text.


Installation
------------

.. code-block:: shell

    pip install leet
    
    
Command-line usage
------------------
.. code-block:: shell

    leet Hello, world!
    > H3ll0, w0rld!
    
    
Library usage
-------------
.. code-block:: shell

    from leet import leet
    
    print(leet('Hello, world!'))
